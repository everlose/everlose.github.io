CSS flex布局实践

发表于 2017-03-16 更新于 2019-11-09 分类于 css
两列，三列，一行多列，垂直居中，水平居中，圣杯布局，流式布局等经典练习

Basic

flex

flex的默认值为0 1 auto，后两个属性可选，最常用的就是flex-grow。

flex-grow在分配了剩余空间的分配比例，为0代表不分配。

flex-direction

justify-content

属性定义了项目在主轴上的对齐方式

align-items

属性定义了项目在侧轴上的对齐方式

常用布局

两列布局其一

一列固定，另一列适配父容器宽。

两列布局其二

两列往两边靠的布局，不固定宽

三列布局

两列固定尺寸而中间列适应父容器的布局

一行多列均等分

一行不固定列数，并且要求都能均等分分布的布局

固定操作栏

顶部和底部的操作栏固定不跟随内容滚动

水平垂直居中

弹窗布局就是一个常见的元素水平垂直居中的场景

流式布局

不定行的布局，要求每个元素固定大小，常见于选择

# css 3
自适应等比例多列布局
