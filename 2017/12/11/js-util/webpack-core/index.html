其他引入方式

就如 webpack 3 官网所说

对比 Node.js 模块，webpack 模块能够以各种方式表达它们的依赖关系，几个例子如下：

  • ES2015 import 语句
  • CommonJS require() 语句
  • AMD define 和 require 语句
  • css/sass/less 文件中的 @import 语句。
  • 样式(url(…))或 HTML 文件中的图片链接(image url)

webpack 能处理的模块类型比较多，上面的例子里我们只演示 ES2015 import 语句。
